| Category | Use For |

## Bug Reports

If you've found a reproducible bug, open a [GitHub Issue](https://github.com/your-org/vaultflow/issues/new?template=bug_report.yml) using the Bug Report template.

Please search existing issues before opening a new one — your bug may already be reported and have a workaround.

## Security Issues

**Do not open public GitHub issues for security vulnerabilities.**

Email **security@vaultflow.io** for responsible disclosure. See [SECURITY.md](SECURITY.md) for full details.

## Response Times

| Channel | Expected Response |
|---|---|
| Security email | 48 hours |
| GitHub Issues (bugs) | 3–5 business days |
| GitHub Discussions | Best effort (community-driven) |
| Pull Requests | 3 business days for initial review |
